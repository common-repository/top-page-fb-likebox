This plugin will enable top page button and facebook likebox in your wp site. You can change color and other important settings from Option Panel.

== Description ==
This plugin will help you top page and facbook likebox show for front page. When any user your website scrolling down automatically show top button for go top of website also facebook likebox showing with minimize and maximize button. If you want to change background color, font size, border, border radius, button color etc you can change easily any time from admin panel. 

= Plugin Features =

* Very Lightweight. Only 51.8 KB.
* Unlimited Color
* Enable and Disable Options
* Important Settings Options Panel (BG Color, Font Size, Border etc You will can change easily. ) 
*  Nice Style & Very Fast

== Installation ==

1. Upload top-page-fb-likebox ' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. When the plugin will be completely active, you will see 'Top Button' in the left menu. Just click the menu and enjoy.
